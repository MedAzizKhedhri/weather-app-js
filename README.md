# 🌤️ Tunisian Weather App

A beautiful and responsive weather application that displays real-time weather conditions for major Tunisian cities. Built with pure JavaScript, HTML, and CSS - no frameworks required!

![Weather App](https://img.shields.io/badge/Version-1.0-blue)
![GitHub](https://img.shields.io/github/license/MedAzizKhedhri/weather-app-js)
![GitHub last commit](https://img.shields.io/github/last-commit/MedAzizKhedhri/weather-app-js)

## ✨ Live Demo

🔗 **Try it now:** [Live Weather App](https://medazizkhedhri.github.io/weather-app-js/)

## 🎯 Features

- **🏙️ Tunisian Cities Focus** - Weather data for 8 major Tunisian cities
- **🔍 Real-time Search** - Look up weather for any city worldwide
- **📱 Fully Responsive** - Perfect experience on desktop, tablet, and mobile
- **⚡ Lightning Fast** - Pure vanilla JavaScript for optimal performance
-  
- **🎨 Beautiful Design** - Clean, modern interface with intuitive weather icons
- **🔒 100% Secure** - No API keys required, uses free Open-Meteo API
- **🌍 Open Source** - MIT licensed - learn, modify, and share freely

## 🚀 Quick Start

### Option 1: Use Online
Visit the [live demo](https://medazizkhedhri.github.io/weather-app-js/) - no installation needed!

### Option 2: Run Locally
```bash
# Clone the repository
git clone https://github.com/MedAzizKhedhri/weather-app-js.git

# Navigate to project directory
cd weather-app-js

# Open in your browser (or simply double-click index.html)
open index.html
```

## 🎮 How to Use

1. **🌡️ View All Cities** - App loads weather for all Tunisian cities automatically
2. **🔍 Search Any City** - Use the search bar to find weather worldwide
3. **🔄 Refresh Data** - Click refresh button for latest weather updates
4. **🏠 Show Tunisian Cities** - Click "Show All Tunisian Cities" to return home

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Weather API**: [Open-Meteo](https://open-meteo.com/) - Free & reliable
- **Icons**: Unicode weather emojis
- **Hosting**: GitHub Pages
- **License**: MIT
